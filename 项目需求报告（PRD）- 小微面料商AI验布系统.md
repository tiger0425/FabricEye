# 小微面料商AI验布系统 - 项目需求报告（PRD）V2.0

> **文档版本**: v2.0  
> **更新日期**: 2026年3月3日  
> **文档状态**: 已实现 MVP，进入产品化阶段  
> **上一版本**: v1.0

## 主要变更

### V2.0 已实现功能
- [x] CascadeEngine 级联检测引擎（Flash + Plus 双阶段）
- [x] 视频回放与缺陷标记系统
- [x] 视频导出功能（FFmpeg 生成标记视频）
- [x] 级联状态实时监控面板
- [x] 成本优化 72%（级联架构降低 API 调用成本）

### 技术架构升级
- 从单一 API 调用升级为级联双检架构
- 从双路处理升级为三线程并行（采集/Flash/Plus）
- 新增智能去重（IoU + 时间窗口）
- 新增实时 WebSocket 推送

### 成本优化
- 单卷检测成本从 22.5元 降至 6.3元（降低 72%）
- 月度运营成本控制在 425-929元
